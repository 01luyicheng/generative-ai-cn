## 产品

### Vertex AI

- 使用 [Vertex AI](https://cloud.google.com/vertex-ai) 与基础模型交互、定制并将其嵌入到您的应用中
- 在 [Model Garden（模型花园）](https://cloud.google.com/model-garden) 中访问基础模型
- 在 [Vertex AI Studio](https://cloud.google.com/generative-ai-studio) 上通过简洁的界面微调模型
- 使用 [Vertex AI Search](https://cloud.google.com/products/agent-builder) 构建和部署基于数据的 AI 智能体
- Google 基础模型
  - [Gemini API 模型](https://cloud.google.com/vertex-ai/generative-ai/docs/learn/models#gemini-models)
  - [Imagen API 模型](https://cloud.google.com/vertex-ai/generative-ai/docs/learn/models#imagen-models)
  - [MedLM API 模型](https://cloud.google.com/vertex-ai/generative-ai/docs/learn/models#medlm-models)

### Gemini 模型

- [Gemini](https://cloud.google.com/vertex-ai/docs/generative-ai/multimodal/overview) 是 Google DeepMind 推出的多模态模型，能够理解几乎任何输入，融合不同类型的信息，并生成几乎任何形式的输出。
- 使用文本、图像、视频或代码在 [Vertex AI](https://console.cloud.google.com/freetrial/?redirectPath=%2Fvertex-ai%2Fgenerative%2Fmultimodal%2Fcreate%2Ftext) 中对 Gemini 进行提示和测试。

### Google Cloud 中的 Gemini

- [Gemini Code Assist](https://cloud.google.com/products/gemini/code-assist) 提供 AI 辅助能力，帮助开发者在 VS Code、JetBrains 等主流代码编辑器以及 Firebase 等开发平台上以更高的效率、质量和安全性构建应用。
- [Google Cloud 的 Gemini](https://cloud.google.com/products/gemini) 系列产品帮助用户更高效地工作和编码、深入洞察数据、应对安全挑战等。
